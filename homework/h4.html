  1. What is the Box Model? Browser sees every HTML element as a rectangular box. Css box model describes how the size of elements are calculated.
  2. What does it mean to be a "block-level" element? Block-level elements by default are the same height as the content contained between the tags, but they span the entire width of its container, even if the content itself doesn't. Block level element always start on a new line, stack on top of each other.
  3. Provide one tag that is, by default, rendered as a block-level element paragraph p.
  4. What is an inline element? Height and width = content. The elements align to the left side by side in a line
  5. Provide one tag that is, by default, rendered as an inline element. span tag.
  6. What are the two ways we should include CSS in our pages? Inline, internal and external CSS.
